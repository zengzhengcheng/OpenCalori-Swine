# 后端接口说明文档

> **数据处理系统** — 前后端对接指南 v2.0  
> 本文档说明了项目的模块架构、控件清单、信号槽接口以及对接方式。

---

## 一、项目架构

```
main.py                 ← 入口，组装各模块
  ├── ui_loader.py      ← 加载 mainwindow.ui
  ├── qss_styles.py     ← QSS样式（可独立修改）
  ├── icon_manager.py   ← SVG图标加载
  ├── language_manager.py ← 语言切换管理
  │     └── translations.py ← 翻译字典
  └── slot_handlers.py  ← 【后端接口层】槽函数在此实现
```

**客户主要编辑的文件：`slot_handlers.py`**  
所有按钮的业务逻辑入口都在此文件中，每个方法有详细注释。

## 三、后端对接方式

### 方式一：直接修改 slot_handlers.py（推荐）

在 `SlotHandlers` 类中找到对应的槽函数，添加业务逻辑：

```python
# slot_handlers.py 中
def on_select_sw_data(self):
    """选择室外数据文件"""
    file_path, _ = QtWidgets.QFileDialog.getOpenFileName(
        self.ui,
        "选择室外数据",
        "",
        "CSV Files (*.csv);;Excel Files (*.xlsx);;All Files (*)"
    )
    if file_path:
        self.outdoor_data_path = file_path
        self.log_message(f"✅ 已选择室外数据: {file_path}")
```

### 方式二：继承 SlotHandlers 创建子类

```python
# my_backend.py
from slot_handlers import SlotHandlers

class MyBackend(SlotHandlers):
    def __init__(self, ui_window, lang_manager):
        super().__init__(ui_window, lang_manager)
        self.outdoor_data = None

    def on_select_sw_data(self):
        # 覆写：添加自定义逻辑
        file_path, _ = QtWidgets.QFileDialog.getOpenFileName(...)
        if file_path:
            self.outdoor_data = self.load_csv(file_path)
```

然后在 `main.py` 中替换：

```python
# 将这行:
self.slot_handlers = SlotHandlers(self.ui, self.lang_manager)
# 改为:
from my_backend import MyBackend
self.slot_handlers = MyBackend(self.ui, self.lang_manager)
```

### 方式三：使用 get_widget() 直接操作控件

```python
# 获取任意控件
widget = slot_handlers.get_widget("lineEdit_jiange")
value = widget.text()

# 向日志面板追加消息
slot_handlers.log_message("✅ 处理完成，共120条记录")
```

## 五、翻译扩展方法

在 `translations.py` 中添加新的文本条目：

```python
TRANSLATIONS["zh"]["your_key"] = "你的中文文本"
TRANSLATIONS["en"]["your_key"] = "Your English Text"
```

在代码中获取翻译文本：

```python
text = self.lang.get_text("your_key")
```

## 六、样式修改方法

所有QSS样式集中在 `qss_styles.py` 中。

修改方法：
1. 打开 `qss_styles.py`
2. 找到对应的选择器（如 `QPushButton#pushButton`）
3. 修改属性值
4. 保存后重新运行即可生效

颜色体系说明：
- 主色调（天蓝）: `#0EA5E9`
- 强调色（靛紫）: `#6366F1`
- 辅助色（青色）: `#06B6D4`
- 背景色: `#F0F4F8`
- 卡片色: `#FFFFFF`
